Embedthis BuildTools Roadmap

The EJScript roadmap is created by the accumulated input from all users. As such, it is subject to change. This roadmap is not a commitment to product, but the current thinking about future directions.

Those who shout loudest -- often get what they ask for. To request a feature, please send email to:

dev@embedthis.com

Feature Wish List

This list is not in any particular order.
  • Byte-code interpreter (VM based design)
  • Enhanced inheritance
  • String, Regexp types
  • HTTP support

What's in the Next Release

The next release in the stable branch will be the Embedthis EJScript 2.1 which is currently unscheduled. Patches to the stable branch are released only as issues require it.

For the development branch, we will release incremental technology updates and fixes. The next is 3.0 which is expected in Q4.

Please tell us what you would like to see at dev@embedthis.com.
